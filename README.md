 # 🌍 Travel Explorer Pro

## 📖 Overview

Travel Explorer Pro is a responsive travel and tourism web application built using HTML, CSS, and JavaScript. The application allows users to explore countries, tourist destinations, travel packages, and weather information through a modern and user-friendly interface.

The project uses MockAPI to fetch and manage travel-related data dynamically, making it a practical demonstration of frontend development and API integration.

---

## 🚀 Features

### 🏠 Home Page

* Attractive hero section
* Auto-changing background images
* Featured destinations
* Popular travel packages
* Responsive navigation bar

### 🌎 Countries

* View country information
* Country name, capital, currency, and population
* Dynamic data from MockAPI

### 📍 Places

* Explore famous tourist destinations
* Place images and descriptions
* Search and filter functionality

### 🎒 Travel Packages

* Package Name

### 🌦️ Weather Information

* Weather conditions
* Dynamic weather-based backgrounds

### 🔐 Authentication

* User Registration
* User Login
* Form Validation
* Data Storage using MockAPI

### 📞 Contact Page

* User Feedback Collection
